## Githubについて
### 命名ルール
branch
- server or client/作業名

pull request
- server or client/Trelloのチケット名

commit
- git commit -v
- 1行目にタグ(fix, add, modify, remove)と概要
- 3行目から詳細
- タグ
  - fix：バグ修正
  - add：新規（ファイル）機能追加
  - modify：機能修正（バグではない）
  - remove：削除（ファイル）

## Trelloについて
### Listの役割
- Todo : 行う予定のチケット
- InProgress : 進行中のチケット
- OnHold : ストップ中のチケット
- Done : 完了したチケット

### Cardについて
- チケットには必ずLabelをつけてください
  - Crient : 青色
  - Server : 緑色
  - Design : 黄色
  - Others : 灰色
- 誰がその作業をやるのかMemberも必ずつけてください
